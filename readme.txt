== Description ==

<p><strong>Responsive Wordpresss Testimonial</strong> is an excellent Wordpress plugin to show your client's testimonials in any Wordpress website. This responsive plugin is very lightweight, easy-to-use, and can be placed anywhere in your website using TinyMCE shortcode button. This testimonial plugin has excellent auto cycling and hovering effect with Wow.js & Animate.css</p>

<p>Have a look at the demo <a href="http://azizultex.me/products/responsive-wordpress-testimonial-plugin/" target="_blank">HERE</a></p>

<p><strong>Main Features:</strong></p>
<ul>
<li>ZERO coding skills required.</li>
<li>Lightweight and easy-to-use</li>
<li>Select Unlimited colors using jQuery color picker</li>
<li>Random cycling among all testimonials one by one using LocalScroll Plugin </li>
<li>Many Animation using Wow.js and Animate.css</li>
<li>Social Media Icons with Font awesome</li>
<li>Shortcode button in the editor that will let you use the plugin anywhere</li>
<li>Fully responsive for all devices</li>
<li>Excellent support ensured!</li>
</ul>
