    Getting started

    We assume you have your own WordPress & Theme installed, so let's get started! if not, and if you don't know how to do that, check this link



    1. Installationtop

    There are 2 ways to install wordpress Plugins: upload unzipped theme via FTP or use the zipped file directly by on the wordpress theme installer .

    1.1 Upload via FTP
    • Log into your site via FTP . (if you don't have any FTP program you can use FileZilla )
    • Browse to your folder located at yoursite.com/wp-content/plugins
    • Upload the un-zipped theme folder
    • Login to Wordpress, in Left Menu go to Plugins
    • Find & Activate the Plugin



    1.2 Upload via Wordpress
    • Login to Wordpress
    • go to Plugins
    • you will see Add New tabs. Click on the tab
    • Thenclick Upload right below the tab
    • Click choose file, browse to theme folder and choose the file responsive-wordpress-testimonial-installable.zip, click Install and wait for a seccond.
    • Activate the plugin



    2. Import & Setup Sample Datatop

    The plugin comes with Sample data (just like the plugin preview) that allow you to create some dummy content to your site to help you get started with the plugin.

    After installation and activation of the plugin, you will see a blank testimonial list view page in your admin panel like below. To see demo testimonials, you need to upload the demo content.





    2.1 How to Import the sample data?
    • You can find the mylocalhostwordpress.wordpress.2014-08-30.xml file in plugin package in the 'demo content' folder.
    • Login to Wordpress, in Left Menu go to Tools -> Import
    • Choose Wordpress, if the plugin haven't not been installed , the popup window will show up and just click Install Now.
    • Click Activate Plugin & Run Importer
    • Click choose and browse to the mylocalhostwordpress.wordpress.2014-08-30.xml
    • Click upload file and import,
    • Wait a few seconds .


    2.2 After Importing Data

    After importing data go to My Testimonials ->All Testimonials. You will see the testimonails with demo client's image. Now you have to add your client's image (featured images).



    3. Using the Plugintop




    3.1 Adding Single Testimonial
    • If you have imported the demo content hypercoderbdplugindirectory.wordpress.2014-08-23.xml file, you can delete them completely and create new testimonials. Or edit them to with your client image, testimonial, and their name & company name.
    • Add your testimonial title (1), featured image (2), testimonial (3), and client's info (4) as shown in below screenshot


    3.2 Inserting Testimonial
    • Now that you have added your testimonials and want to use it on your page. So, go to your desired page and edit it.
    • When you want to insert your testimonial just click on the icon and a popup window with the setting will be open.
    • In the Testimonial Heading Section, put your Testimonial Heading, Font Size, Heading Color, and Heading Background.
    • In the Testimonial Setting Section, put your Testimonial Quote Color, Background Color, and your quote animation style.
    • In the Client Info Setting Section, put your Client Name Color, Company Name Color, Background Color, Animation Style, Social Icon Color, and Social Icon hovering color.
    • In the Image Setting Section, put your image rotate degree, Image Animation Type, set Yes or No for client's round image, and set Borders if you want. Default Image border is 0px solid #eee where 0px is the border width, solid is border style, and #eee border color. You can learn more about border here: http://www.w3schools.com/css/css_border.asp
    • When you are done with your setting. Click on Insert Testimonial to get your shortcode in editor

    For selecting any UI color, you may visit at : FlatUIColors, for Animation Selection help visit Animate.css
